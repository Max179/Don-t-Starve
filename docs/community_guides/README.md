# 饥荒中文社区攻略资讯库

审计日期：2026-07-18（Asia/Shanghai）

本目录整理中文社区攻略来源，定位是“玩法路线、经验判断和资料发现层”，不替代官方记录、Wiki 数据库或精确机制表。需要精确数值、掉落、配方、版本改动时，优先复核本仓库的 `data/dont_starve_wiki.sqlite`、Klei/Steam 官方记录和主流 Wiki。

## 文件

- `source_catalog.md`：按平台整理的来源卡片、可信度、适用主题和复核提醒。
- `topic_index.md`：把来源整合到可查询的攻略主题索引。
- [data/community_guides_seed.json](../../data/community_guides_seed.json)：同一批来源的结构化种子数据，便于后续导入数据库。

## 数据库表

已通过 `scripts/import_community_guides.py` 导入 `data/dont_starve_wiki.sqlite`：

- `community_guide_sources`：21 条来源卡片。
- `community_guide_topics`：77 条来源主题标签。
- `community_guide_topic_index`：5 条主题推荐索引。

## 可信度标尺

- `A`：长文/系列化/高互动，作者或平台信号清晰，适合作为主攻略入口；仍需核验版本变化。
- `B`：主题明确、可引用、能补足某个玩法模块；适合和其他来源交叉验证。
- `C`：论坛讨论、旧版本、单点技巧或平台聚合；只能作为线索，不直接作为事实源。
- `blocked`：入口存在但当前无法审计具体内容，等待登录态、人工链接或后续采集。

## 平台状态

- Bilibili：专栏/动态页可直连；部分视频页在脚本抽查中返回 412，但浏览器和搜索结果仍可定位，适合收录实操演示、地图/洞穴/种田图解。
- 知乎：搜索结果和专栏入口可定位；脚本直连专栏页返回 403/安全验证，适合先做来源索引，正文细节需浏览器或人工复核。
- 游民星空：专题页和攻略页可直连，适合做中文站点索引和旧攻略交叉参考。
- TapTap：论坛攻略和合集页可直连，适合新手时间轴、移动端/玩家经验和社区修订记录。
- 小红书：搜索页可访问，但当前只返回前端应用壳，未能获得可审计的具体笔记链接；已保留待补采入口。

## 使用原则

1. 先看 `topic_index.md` 找主题，再回到 `source_catalog.md` 查看来源可信度。
2. 社区攻略只记录“建议”和“路线”，精确数值统一复核数据库或官方/Wiki。
3. 区分《饥荒》单机、《饥荒：联机版》和手游/合辑版，避免把版本差异混用。
4. 不复制长文或搬运图片；保留标题、作者、链接、抓取日期、摘要和复核备注。
