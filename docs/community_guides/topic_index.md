# 攻略主题索引

审计日期：2026-07-18（Asia/Shanghai）

## 新手入门路线

推荐阅读顺序：

1. 视频入门：[Bilibili - 亚丹Ardent 萌新终极入门攻略](https://www.bilibili.com/video/BV1n8411d7SL/)
2. 文字总目录：[知乎 - 饥荒联机版大型攻略目录](https://zhuanlan.zhihu.com/p/156956476)
3. 时间轴流程：[TapTap - 新手向全程攻略](https://www.taptap.cn/moment/156969472316409026)
4. 清单化补充：[Bilibili - 任务化新手教程](https://www.bilibili.com/read/cv16443362/)

可沉淀的资讯条目：

- 开局优先级：照明、基础采集、探图、金块和一本/二本科技。
- 建家节点：先满足烹饪、储物、防雷、常用资源移植，再扩展蜂箱/鸟笼/农田。
- 时间轴：1-10 天探图，10-15 天建家，15-20 天补食物系统，入冬前准备保暖和战斗物资。
- 复核项：世界设置、洞穴开启建议、季节天数、首个 Boss 出现逻辑和最新角色技能树。

## 四季生存

主来源：

- [知乎 - 生存篇](https://zhuanlan.zhihu.com/p/156964823)
- [TapTap - 新手向全程攻略](https://www.taptap.cn/moment/156969472316409026)
- [Bilibili - 新手攻略万字长文](https://www.bilibili.com/opus/926001627708522512)

资讯条目：

- 秋季：探图、科技、锅、冰箱、基础护甲、稳定食物链。
- 冬季：保暖、海象/象牙、首个季节 Boss、避免在基地接战。
- 春季：防雨、防雷、潮湿管理、青蛙雨和春季 Boss。
- 夏季：过热、自燃、沙漠/绿洲或洞穴路线、灭火器与吸热火。

复核项：联机版与单机版护甲机制、季节 Boss 触发、夏季建筑自燃设置和世界配置。

## 食物与恢复

主来源：

- [知乎 - 食物篇](https://zhuanlan.zhihu.com/p/160941435)
- [游民星空 - 实用食谱及推荐做法](https://www.gamersky.com/handbook/201612/847156_3.shtml)
- [TapTap - 新手需要知道一些东西](https://www.taptap.cn/moment/640156117896267673)

资讯条目：

- 萌新常备：肉丸解决饱食，波兰水饺解决回血，培根煎蛋/肉汤等作为进阶选择。
- 资源链：怪物肉、蛋、蔬菜、冰和浆果是早期食谱学习重点。
- 设施链：锅、冰箱、晾肉架、鸟笼是从“捡食物”过渡到“可持续供给”的关键。

复核项：配方优先级、食材度、烹饪禁忌、保质期和当前版本数值应从数据库或 Wiki 读取。

## 种田与巨大作物

主来源：

- [游民星空 - 巨大作物种植攻略](https://www.gamersky.com/handbook/202206/1487506.shtml)
- [知乎 - 巨大农作物种植搭配技巧](https://zhuanlan.zhihu.com/p/429409488)
- [Bilibili - 巨大果实种植方法](https://www.bilibili.com/opus/1111623753429680169)

资讯条目：

- 入门设备：耕地机、园艺锄、水壶、耕作先驱帽。
- 核心机制：作物家族、三种养分、水分、杂草、对话/快乐度。
- 资料库建议：为每种作物记录季节偏好、养分消耗/释放、推荐混种组合和是否适合巨大化。

复核项：Bilibili 巨大果实笔记时间显示为 2025-09-12，当前已是 2026-07-18，可以收录但仍需核对版本；所有养分表应对照 Wiki 或本地数据库。
